# employee-attrition
A machine learning web app that predicts employee attrition risk.

# Problem
HR teams struggle to identify employees at risk of leaving before its too late.

# Solution
ML model trained on IBM HR Analytics dataset to predict attrition probability

# Models used
- Logistic Regression (ROC-AUC: 0.759, better recall)
- Random Forest (ROC-AUC: 0.771, better accuracy)

# Tech Stack
Python, Scikit-learn, Streamlit, Pandas, Joblib
